# 深度搜索

深度搜索模式在开启后，会通过特定关键词触发，生成任务图并调用多个子agent并行完成任务，最后进行总结。该模式会占用大量时间和资源，一般情况下不建议开启。

## 功能介绍

深度搜索模式采用多agent协作的方式处理复杂任务：

1. **任务图生成**：系统根据你的需求自动生成任务分解图
2. **多agent并行**：调用多个子agent同时处理不同的子任务
3. **结果汇总**：所有子任务完成后，系统会自动总结并整合结果

## 使用方法

### 1. 开启深度搜索

在对话设置菜单中，打开"深度搜索"开关即可启用。

![深度搜索开关](/manuals/assets/chat/summary/89624e415e2ac0ed7a059a15d8bcd043.jpg)

### 2. 关键词触发

开启深度搜索后，当对话中包含以下触发词时，系统会自动启用深度搜索模式：

**常用触发词：**

```
分析、比较、研究、调查、总结、评估、计划、设计、开发
详细、全面、深入、系统、综合、多角度、步骤、方案
分几个、多个方面、详细解释、具体分析、如何实现、实施方案
```

## 使用场景

深度搜索模式适合以下场景：

- **信息搜集**：需要从多个角度搜集和整理信息
- **复杂分析**：需要对某个主题进行深入、全面的分析
- **方案设计**：需要系统性地设计解决方案或实施计划
- **对比研究**：需要比较多个选项或方案

**注意事项：**
- 深度搜索会占用大量时间和计算资源
- 建议仅在需要全面、深入分析时使用
- 对于简单问题，使用普通模式即可
